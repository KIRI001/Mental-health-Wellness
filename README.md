# MHW Repository

## Weather App (CSS path/casing fix)

The Weather App is in `/weather-app` and uses:

- HTML: `/weather-app/index.html`
- CSS: `/weather-app/style.css`
- JS: `/weather-app/script.js`

To verify styles load correctly, open `/weather-app/index.html` in a browser and confirm the gradient background and card UI appear.

Before searching weather, set an API key in the browser console:

`window.OPENWEATHER_API_KEY = "YOUR_API_KEY"`
