# 📌 Overview

Mental health has become one of today's most important challenges. **MindCare** is designed to create a peaceful digital space where users can manage their mental wellness, connect anonymously with others, and access wellness-focused features through an intuitive and responsive interface.

The platform combines a modern SaaS-inspired design with interactive features to improve user engagement and accessibility.

---

### 💊 Medicine Reminder

- Create reminder schedules for medicines with custom dosage and times
- Scheduled in-app toast notifications triggered when a reminder is due via a background runner

---

### 📘 Private Journaling

- Write anonymous personal reflections securely
- Entries are pseudonymized using SHA-256 hashes to guarantee user privacy

# ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/KIRI001/Mental-health-Wellness.git
```

Move into project

```bash
cd Mental-health-Wellness
```

Install dependencies

```bash
npm install
```

Run server

```bash
node server.js
```

Open

```
http://localhost:3000
```
